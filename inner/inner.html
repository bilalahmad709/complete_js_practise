innerText Vs innerHTML


what is innerText:

In JavaScript, innerText is a property of an HTML element that allows you to get or set the text content inside that element, excluding any HTML tags. Unlike innerHTML, which returns all the HTML within an element, innerText returns just the visible text, as displayed by the browser.
