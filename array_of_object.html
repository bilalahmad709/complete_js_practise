What is Array of object :

an array of objects is a data structure that holds multiple objects within a single array variable. Each element in the array is an object, allowing you to store and access multiple objects in a sequential manner.
For example :

// Array of objects
let cars = [
{ make: 'Toyota', model: 'Corolla', year: 2020 },
{ make: 'Honda', model: 'Civic', year: 2019 },
{ make: 'Ford', model: 'Mustang', year: 2021 }
];
// Accessing elements in the array
console.log(cars[0].make); // Output: Toyota
console.log(cars[1].model); // Output: Civic
console.log(cars[2].year); // Output: 2021
